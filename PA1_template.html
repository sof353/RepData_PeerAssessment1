4. Imputing missing values

Duplicate the dataset to create a new dataset. If data in steps variable contains NAs then the avergae value for that interval will be substituted into the dataset in place of the NA. Identify the NAs and substitute with the corresponding interval averge.

dat2 <- dat1
for(i in 1:nrow(dat2)){
      if(is.na(dat2$steps[i])){
            dat2$steps[i] <- meanStepsByInt$mean[meanStepsByInt$time == dat2$time[i]]
      }
}
